LAB LITERALES DE PYTHON Y CADENAS

En esta parte empecé a ver qué son los literales, que básicamente son los datos que uno escribe directamente en el código. Aprendí que para Python un 2 solito es un número para hacer cuentas, pero un "2" con comillas es solo un texto y no valen lo mismo. Me pareció muy útil lo de los guiones bajos para escribir números largos como 10_000_000 porque así no me pierdo contando ceros. También vi que existen los flotantes que llevan punto decimal y los booleanos que son solo True o False para decir si algo es verdad o mentira. Lo que más me costó pero ya le agarré el tiro fue lo de las comillas, que toca usar la barra inclinada o cambiar de comillas simples a dobles para que el programa no se confunda cuando quiero imprimir un texto que ya lleva comillas adentro.
