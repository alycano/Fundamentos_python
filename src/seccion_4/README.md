LAB 1 VARIABLES

En esta sección aprendí a usar las variables, que son como cajitas donde guardo información para usarla más tarde. Descubrí que ponerles nombre tiene sus reglas, como que no pueden empezar con números y que Python es muy serio con las mayúsculas: no es lo mismo la variable manzanas que Manzanas. Lo que más me gustó fue ver que puedo cambiar el valor de una variable usando el mismo nombre, como cuando sumé las manzanas de Juan, María y Adán para sacar un total. También aprendí unos trucos para escribir menos código, como poner += en vez de repetir el nombre de la variable, lo que hace que todo se vea más limpio y profesional.
